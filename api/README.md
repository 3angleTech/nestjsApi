## Installation

```bash
$ npm install
```

Add ```127.0.0.1 3at-nest-postgres``` to ```/etc/hosts``` so you can run the app locally also, not only in docker container.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
